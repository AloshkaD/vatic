The database can be reset with:
    $ turkic setup --database --reset

Videos can be extracted with:
    $ turkic extract /path/to/video.mp4 /path/to/output/directory
    $ turkic extract /path/to/video.mp4 /path/to/output/directory --no-resize
The second will not receive the frames.

Videos can be loaded with:
    $ turkic load identifier /path/to/output/directory --offline --length=9000 Person Car Bicycle
The last arguments specify object categories. --length specifies how long (in frames) each shot should be.

You can then get a list of URLs with:
    $ turkic publish --offline
You can send these to workers, etc.

You can get all data with:
    $ turkic dump identifier --matlab -o annotations.mat 
    $ turkic dump identifier --matlab -o annotations.mat -i
The first command only gives ground truth. The second will interpolate.

You can visualize the results with:
    $ turkic visualize identifier tmp/ -i
